# Button

Component to creating button.

```ts
import { Button } from "eduprog-ui";
```

## Examples

### Button view

To change the appearance of the button, set the `view` to one of the following values:
`"default"`, `"action"`, `"link"`, `"pseudo"`, `"clear"`, `"raised"`.

<Canvas withSource="open">
  <Container adaptive>
    <Row spacing={1}>
      <Column>
        <Button view="default">Button</Button>
      </Column>
      <Column>
        <Button view="action">Button</Button>
      </Column>
      <Column>
        <Button view="link">Button</Button>
      </Column>
      <Column>
        <Button view="pseudo">Button</Button>
      </Column>
      <Column>
        <Button view="clear">Button</Button>
      </Column>
      <Column>
        <Button view="raised">Button</Button>
      </Column>
    </Row>
  </Container>
</Canvas>

### Button size

To resize the button, set the the `"size"` property to one of the following values: `"xl"`, `"lg"`, `"md"`, `"sm"`, `"xs"`.

  </div>
</Canvas>
