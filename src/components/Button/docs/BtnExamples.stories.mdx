## Примеры

### Тип кнопки

Чтобы изменить вид кнопки, установите свойство `view` в одно из следующих значений:
`"default"`, `"action"`, `"link"`, `"pseudo"`, `"clear"`, `"raised"`.

<Row>
  <Column ml={5} mr={1}>
    <Button type="default">Button</Button>
  </Column>
  <Column ml={1} mr={1}>
    <Button type="action">Button</Button>
  </Column>
  <Column ml={1} mr={1}>
    <Button type="link">Button</Button>
  </Column>
  <Column ml={1} mr={1}>
    <Button type="pseudo">Button</Button>
  </Column>
  <Column ml={1} mr={1}>
    <Button type="clear">Button</Button>
  </Column>
  <Column ml={1} mr={1}>
    <Button type="raised">Button</Button>
  </Column>
</Row>
