## Примеры

### Тип кнопки

Чтобы изменить вид кнопки, установите свойство `view` в одно из следующих значений:
`"default"`, `"action"`, `"link"`, `"pseudo"`, `"clear"`, `"raised"`.

<Container adaptive>
    <Row>
        <Column col={2}>
            <Button view="default">Button</Button>
        </Column>
        <Column col={2}>
            <Button view="action">Button</Button>
        </Column>
        <Column col={2}>
            <Button view="link">Button</Button>
        </Column>
        <Column col={2}>
            <Button view="pseudo">Button</Button>
        </Column>
        <Column col={2}>
            <Button view="clear">Button</Button>
        </Column>
        <Column col={2}>
            <Button view="raised">Button</Button>
        </Column>
    </Row>
</Container>
