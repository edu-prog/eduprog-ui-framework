# Colours

## Default Colours

Eduprog UI provides a set of standard colours that are ubiquitous in component organisation

Basic colours can be changed and customised to each developer's taste for a more personalised application

## Customize Theme Colours

You can change framework variables like any other css variables

```css
:root {
  --ep-color-primary: #6320ee;
  --ep-color-secondary: #8075ff;
  --ep-color-accent: #50c878;
  --ep-color-success: #46c93a;
  --ep-color-danger: #a00707;
}
```
